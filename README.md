## 📘 Sobre o Projeto

Esta é uma Calculadora de IMC desenvolvida como um projeto pessoal para demonstrar minhas habilidades em HTML, CSS e JavaScript. O objetivo é oferecer uma ferramenta simples e eficiente para calcular o Índice de Massa Corporal (IMC) de forma rápida e intuitiva.

Link para a Calculadora: [Calculadora de IMC](https://lerraji-sousa.github.io/Calculadora-de-IMC/)

<br>

## 🔨 Tecnologias Utilizadas

- **HTML**: Para estruturar a interface da calculadora.
- **CSS**: Para estilizar e dar uma aparência atraente à calculadora.
- **JavaScript**: Para implementar a lógica de cálculo e adicionar interatividade.

<br>

## 🎯 Funcionalidade

- Cálculo automático do IMC com base no peso e altura inseridos.
- Exibição de categorias de IMC conforme os padrões da Organização Mundial da Saúde (OMS).

<br>

## 🚀 Como Usar

1. Acesse o link: [Calculadora de IMC](https://lerraji-sousa.github.io/Calculadora-de-IMC/)
2. Digite seu peso (em quilogramas) e altura (em metros) nos campos apropriados.
- **Observação:** Utilize um ponto ( . ) como separador decimal, em vez da vírgula ( , ).
3. Clique no botão "Calcular" para obter seu resultado de IMC e a classificação correspondente.
